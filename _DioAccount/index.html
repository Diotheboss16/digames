Setup

This app uses Firebase (Firestore). Edit firebase-config.js and paste your Firebase web app config.

Hosting URL: https://diotheboss16.github.io/digames/_DioAccount/
Note: with no sign-in, you must set Firestore rules to allow your family to read/write. If you later enable Auth, add diotheboss16.github.io to Firebase Authorized domains.
